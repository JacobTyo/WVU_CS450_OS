Detailed Description

The pcb structure is a Process Control Block. This block consists of the follwing variables: char name[11] - the name of the Process Control Block int class - the process class (0 = system, 1 = application) int priority - integer between 0 and 9 representing the execution priority with 0 being the highest priority and 9 being the lowest priority. int runningstate - the processes ready(0), running(1), or blocked(2) state. int suspendedstate - the process suspended(1), or not suspended(0) state. char stack[1025] - the stack area of 1025 bytes. unsigned char *stacktop - the pointer to the top of the stack. unsigned char *stackbase - the pointer to the base of the stack. pcb *nextpcb - a pointer to the pcb next in line for execution. pcb *prevpcb - a pointer to the pcb previous in line for execution.
